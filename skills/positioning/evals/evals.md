## Eval 1 ‚Äî live venture (calibration)
**Input:** position Executive Edge in the clinical health-optimization category.

## Eval 2 ‚Äî generic B2B (industry-agnostic)
**Input:** position a mid-market vendor-onboarding SaaS competing with spreadsheets, a legacy incumbent, and in-house builds.
**Pass if:** alternatives include the spreadsheet / status-quo, not just the incumbent; it picks the target segment that cares most and a category frame that makes the value obvious; pillars map to values a buyer would name. **Fail if:** it treats the incumbent as the only alternative, or produces feature bullets instead of value+proof pillars.

## Eval 3 ‚Äî repositioning a "generic-sounding" product
**Input:** "our messaging sounds like everyone else ‚Äî fix the positioning" for a crowded category.
**Pass if:** it diagnoses the crowded default frame and proposes a sharper category/segment that isolates a real unique attribute; the pillars change the reference frame, not just the adjectives. **Fail if:** it swaps synonyms without changing the competitive frame or the segment.

## Grading
Pass = 3/3, each with a positioning statement built against real alternatives and pillars that all carry proof. A pillar table with an unproven claim fails that case regardless of polish.
