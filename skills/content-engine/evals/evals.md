# Evals — `/content-engine`

Grade each on: 3–5 pillars that each map to a named offer rung and carry a proof source; a real 1→N repurposing pipeline; a batched weekly cadence with owners; correct voice and guardrails.

## Eval 1 — stand up the engine
**Input:** build the content engine for Executive Edge OS.
**Pass if:** every pillar maps to a specific offer rung (e.g., freemium→$199, $199→$99/mo) and names a proof source; the pipeline shows one anchor fanning to ≥4 derivatives; a Mon–Fri batched cadence with owners; it leads with the wedge pillar. **Fail if:** pillars are generic themes with no offer rung, or it proposes daily net-new posting across many channels.

## Eval 2 — resist the many-fronts pull
**Input:** "I want to post daily on LinkedIn, X, Instagram, TikTok, YouTube, and a newsletter for EE."
**Pass if:** it consolidates to one anchor channel + a repurposing pipeline, names the too-many-fronts failure mode, and shows how the six surfaces get fed from one anchor. **Fail if:** it produces six independent net-new content plans.

## Eval 3 — ideas, not a dump
**Input:** "Give me content ideas for EE."
**Pass if:** ideas are organized under offer-mapped pillars with the metric each serves, not an unstructured list; voice is clinical/premium; no medical claims. **Fail if:** it returns a flat idea list with no pillar/offer structure, or uses banned words ("journey", "wellness", "hack").

## Eval 4 — industry-agnostic (non–Executive-Edge)
**Input:** build the content engine for a premium hospitality group (Ethos: multi-concept restaurants + hotel F&B).
**Pass if:** pillars map to that venture's ladder rungs (reservation/cover → loyalty/membership → private events) each with a proof source; one anchor → ≥4 derivatives; a batched weekly cadence with owners; NO clinical/health-tech assumptions leak in; voice matches the venture. **Fail if:** it imports Executive Edge's clinical framing, or pillars don't map to that venture's offers.

## Grading
Pass = 4/4. Any output where a pillar has no offer rung, or that recommends net-new-per-channel over capture-once/atomize, is an automatic fail — the engine exists to compound toward revenue from one anchor.
