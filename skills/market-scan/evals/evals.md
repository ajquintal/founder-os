# Evals — `/market-scan`

Grade each on: correct structure (market read + table + pricing + wedge + sources), every figure sourced, and a wedge grounded in a real gap.

## Eval 1 — known market (calibration)
**Input:** market scan for clinical health-optimization software (context: Executive Edge).
**Pass if:** market size/growth carry sources; table includes the real leaders (Whoop / Function / Marek / Noom class) with a weakness each; wedge = closed-loop execution + blood-device fusion; names the "will they pay for execution vs insights" risk. **Fail if:** unsourced numbers, or a generic "be better" wedge.

## Eval 2 — narrow niche
**Input:** market scan for "AI intake + care-plan software for independent med-spas."
**Pass if:** it finds the adjacent players (practice-mgmt / EHR / white-label telehealth), admits where hard data is thin (labels provisional), and still lands a defensible wedge. **Fail if:** it fabricates a precise TAM for a niche with no public sizing.

## Eval 3 — head-to-head
**Input:** compare two named competitors on price, model, and positioning for a pricing decision.
**Pass if:** a clean side-by-side with real prices/models sourced, and an explicit "where this venture should sit and why." **Fail if:** it compares features but never lands the pricing implication.

## Grading
Pass = 3/3 with sourced figures and a grounded wedge. A brief with any unsourced asserted number fails that case regardless of polish.

**Web-access assumption:** Evals 1–3 assume web access is available — grade the sourced-number requirements against that. **Provisional / no-web PASS path:** when web is unavailable, a run that labels every figure provisional/unverified, says web was unavailable, and fabricates nothing PASSES — the bar is honest non-fabrication, not numbers it could not fetch. A no-web run that asserts any figure as fact fails.
