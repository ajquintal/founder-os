# /seo-geo — SEO + Generative Engine Optimization

One entity-led plan to rank in search and get cited in AI answers: map → content → technicals. Reads `/market-scan` (entities, competitors) and `/offer-architect` (rungs to map intent to).

## Method (full detail + worked example in `references/method.md`)
1. **Define the entity.** Brand, the category it wants to own, ICP, and the actual questions the ICP types into search *and* asks an AI engine. Entity clarity is the shared root of SEO and GEO.
2. **Map entities + keywords.** Cluster by intent (informational / commercial / transactional / navigational); add conversational AI-answer queries; map each cluster to an offer rung.
3. **Pick one beachhead cluster.** Score intent-value × winnability × strategic fit. Own one cluster before widening. (Guards: too many fronts.)
4. **Plan pillar + cluster content.** One pillar page per cluster + supporting articles, internally linked — engineered for depth (rank) and citation (GEO).
5. **Add the GEO layer.** Direct-answer blocks, quotable stats with sources, comparison tables, FAQ/HowTo schema, and consistent entity data across the web so engines trust and cite you.
6. **Technical checklist.** Crawl, index, sitemap, schema, canonical/robots, Core Web Vitals, llms.txt — each with a done-criterion.

## Rules
- Drafts only — content is drafted for review, never auto-published.
- Own one cluster before widening; depth beats a thin spread.
- GEO ≠ keyword stuffing: earn citations with quotable, sourced, structured answers.
- Voice per `founder-profile/voice-and-brand.md`; no banned words. LYV firewall on content/entities.
- **Claims (G-claims):** on public pages, any Rx / peptide / GLP or health claim must be compliance-reviewed (LegitScript-grade) — never state medical claims or "physician review" / "medical team". Route any clinical claim to review before publish.
- **Roster/dashboard (G-dashboard):** never imply or promise a live team/cohort dashboard pre-contract — roster reporting is a quarterly report until a contract is signed.
